# Project-Yandex-Lyceum
## Мой проект в Яндекс лицее. Преобразование звука в ноты.
### Функции:
+ Загрузка звукового файла
+ Преобразование звукового файла в ноты
+ Просмотр и сохранение нот
+ Редактирование нот(с помощью открытия дополнительного окна, таблицы с чекбоксами в ячейках(выбор ноты, октавы, длительности(?)
+ Открытие нового окна
+ Отмена действия
+ Очистка содержимого

**Дополнительные функции**
+ Создание нот без звукового файла(тем же способом, как редактирование
+ Прослушавание файла
+ Настройка
    * Языка
    * Громкости
    * Заканчивать работу при закрытии окна
    * ...
+ (Завершение работы(до завершения работы все изменения будут сохраняться))?

### Дизайн
- Главное окно с кнопкой для загрузки файла, названием файла, кнопкой переключения окон, кнопкой настроек, меню(кнопка нового окна, очистки содержимого, прослушивание файла, загрузки файла, конца работы)
- окно нот с графическим представлением нот, кнопкой редактирования, создания и сохранения нотб переключения между окнами
- открывающееся окно настроек
- открывающееся окно редактирование/создания нот с таблицей, элементами которой являются чекбоксы со значениями октавы, ноты и длительности, кнопкой сохранить
- открывающееся перед созданием нот окно, с настройкой количества нот(диалог для цифр)
### Архитектура
1. Модуль главного окна
2. Модуль окна нот
3. Модуль рассчёта нот
4. Модуль отрисовки нот
5. Модуль настройки
6. Модуль редактирования нот
7. Модуль создания нот(запускающий модуль редактирования после настройки количества нот)
#### TXT
В тхт будут храниться промежуточные изменения, настройки
#### SQL
В базе данных будут храниться конечные результаты каждой генерации.  
Будет главная таблица содержащая список всех генераций и по одной таблице на генерацию(?)
